## Whitespace

    * indent with spaces
    * indent size is 4 space
    * never mix spaces and tabs

## Syntax


### Parens, Braces and Linebreaks

Always include a blank line above if statements and comments

```javascript

// we are doing stuff
doStuff();

if (condition) {
    doOtherStuff();
}
```
